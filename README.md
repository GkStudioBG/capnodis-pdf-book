# Capnodis PDF Book - Guía Práctica contra el Gusano Cabezudo en Almendro

Landing page para venta de guía digital en PDF sobre manejo integrado del gusano cabezudo (Capnodis tenebrionis) en almendro.

## Stack
- HTML/CSS/JS (vanilla)
- Stripe para pagos
- Email automatizado para entrega de archivos

## Estructura
```
├── index.html              # Landing page principal
├── styles.css              # Estilos
├── script.js               # JavaScript
├── aviso-legal.html        # Aviso legal
├── politica-privacidad.html
├── politica-cookies.html
├── terminos-condiciones.html
└── assets/                 # Imágenes y recursos
```
