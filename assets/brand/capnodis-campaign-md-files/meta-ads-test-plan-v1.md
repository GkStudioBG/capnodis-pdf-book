# Meta Ads Test Plan v1 — Spain Capnodis PDF

Date: 2026-05-26
Market: Spain only
Channel decision: Meta Ads first
Product: `Guía Práctica contra el Gusano Cabezudo en Almendro`
Offer: direct sales / early access preorder

## 1. Test objective

Validate whether Spanish growers/orchard owners respond to a direct paid offer for a practical guide against `gusano cabezudo`.

Primary business question:

> Will the audience click and move toward checkout for a €19.90 early-access guide?

Secondary questions:

- Which pain angle gets attention?
- Which visual hook creates the most qualified clicks?
- Does the audience understand this is a practical guide, not a pesticide/product?
- Do people ask real grower questions in comments/messages?

## 2. Funnel

Meta ad → direct sales landing page → Stripe checkout/preorder.

No lead magnet.
No email-only validation.
No free PDF.

## 3. Campaign setup

### Recommended first campaign objective

Start with either:

1. `Sales` / website purchases if Pixel + conversion tracking are ready.
2. `Traffic` / landing page views if Pixel purchase event is not ready yet.

Recommended:

- If we can install Pixel + checkout event: use `Sales`.
- If not: use `Traffic` for the first smoke test, but judge by checkout clicks and qualitative comments, not just clicks.

### Budget

First test:

- €100-€150 total.
- 5-7 days.
- €15-€25/day.

Do not scale before seeing landing behavior.

## 4. Geographic targeting

Spain only.

Priority regions/ad sets to test:

### Ad set A — Andalucía + Murcia

Reason:

- Strong almond/agriculture relevance.
- Official Andalusian sources confirm problem urgency.

### Ad set B — Aragón + Comunidad Valenciana + Castilla-La Mancha

Reason:

- Relevant almond/stone-fruit production zones.

### Ad set C — Broad Spain rural/agriculture

Reason:

- Let Meta find signals if interest targeting is weak.

If budget is limited, start with:

- One ad set: Spain + agriculture/almond interests.
- Exclude big-city-only delivery if placements skew too urban.

## 5. Audience targeting hypotheses

### Core interest stack

Use Spanish/Meta available equivalents where possible:

- Agricultura
- Agricultura ecológica
- Almendro / almonds / almond farming, if available
- Fruticultura
- Cultivo de almendro
- Agricultor / farming
- Riego por goteo
- Agricultura sostenible

### Broad test

### Important caveat

Meta interests may be imprecise. The creative must self-qualify the audience with clear almond/gusano cabezudo language.

## 6. Creative strategy

Because Meta is interruption-based, the creative must immediately show:

1. Almond tree problem.
2. Drying/decline symptom.
3. Hidden root/larva danger.
4. Practical guide offer.

### Best creative formats for first test

#### Format 1 — Static image with text overlay

Visual:

- Dry/weak almond tree or orchard.
- Text overlay:
  - `¿Tus almendros se están secando?`
  - `El problema puede estar bajo tierra.`

Use when we have suitable images.

#### Format 2 — Simple carousel

Cards:

1. `¿Tus almendros se están secando?`
2. `El daño crítico está en las raíces.`
3. `El momento de actuación importa.`
4. `Guía práctica contra el gusano cabezudo.`
5. `Acceso anticipado: €19.90`

#### Format 3 — Short vertical video / motion graphic

10-20 seconds:

1. Almond tree drying.
2. Root/larva illustration.
3. Calendar/checklist visual.
4. Guide cover mockup.
5. CTA.

This can be produced later; static is enough for first test.

## 7. Creative angles to test

### Angle A — Hidden root damage

Core idea:

`El daño empieza bajo tierra.`

Best for broad pain awareness.

### Angle B — Drying almonds / urgent symptom

Core idea:

`¿Tus almendros se están secando en verano?`

Best for immediate relevance.

### Angle C — Nematodes done wrong

Core idea:

`Antes de comprar nematodos, entiende cuándo funcionan y cuándo fallan.`

Best for growers already considering biological control.

### Angle D — No miracle, integrated plan

Core idea:

`No hay una solución única. Necesitas un calendario de manejo integrado.`

Best for credibility.

Core idea:

Best for new plantations, less urgent for first test.

## 11. Tracking events

Minimum tracking:

- Page view.
- Landing page view.
- CTA click.
- Initiate checkout.
- Purchase/preorder.

If checkout is not ready:

- Track CTA click as micro-conversion.
- Add “notify/interest” only as fallback, not primary validation.

## 12. Comments/messages handling

If comments appear, do not argue or overclaim.

Suggested response style in Spanish:

`Gracias por comentar. La guía no promete eliminar la plaga al 100%, sino ordenar un plan de manejo integrado: diagnóstico, ciclo, humedad del suelo, barreras, nematodos y prevención. Para tratamientos concretos, siempre conviene consultar con un técnico y la normativa vigente.`

Questions to collect:

- `¿En qué zona tienes la parcela?`
- `¿Es secano o regadío?`
- `¿Has visto adultos o solo árboles secándose?`
- `¿Almendro joven o adulto?`
- `¿Has probado nematodos o barreras?`

## 13. Decision rules after test

### Green light

- CTR is healthy for cold Meta traffic.
- Landing page users click CTA.
- Checkout/purchase/preorder exists.
- Comments show real problem awareness.

### Iterate

- Good CTR but weak checkout: improve landing/offer/price/trust.
- Weak CTR: change creative angle/visual.
- Many irrelevant clicks: tighten creative and targeting.
- Comments say “free info exists”: emphasize calendar/decision system and source synthesis.

### Stop or rethink

- No relevant clicks.
- No qualified comments.
- No checkout intent after creative/landing iteration.

## 14. First test recommendation

Run 4 ads:

1. Hidden root damage.
2. Summer drying symptoms.
3. Nematodes angle.
4. Practical calendar / integrated plan.

Use 2 static visual styles:

- Orchard/tree symptom visual.
- Root/larva/calendar educational visual.

Budget:

- Start broad-ish Spain or priority regions.
- Kill worst ads after first 48-72 hours if clearly irrelevant.
