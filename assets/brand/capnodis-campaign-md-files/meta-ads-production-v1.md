## META CREATIVE SPECS

### Carousel Ads (3 carousels × 5 cards = 15 cards)
- **Format:** Single link carousel, 1080×1080px per card
- **Image crop:** Center-cut for non-square sources (crop to 1080×1080 from center)
- **Text overlay:** Max 90 characters in headline, 150 in body
- **CTA:** "Comprar ahora" or "Ver más" (button, not text URL)

### Carousel card crops needed:
- `book-mockup.webp` (717×1115): Crop vertical center → use top portion
- All 16:9 and 3:2 images: Fit to 1080×1080 square with padding or blur background

## META CAMPAIGN SETUP

### Step 1 — Create Campaign
1. Go to Meta Business Manager → Ads
2. Create new campaign
3. Objective: **Sales** (or "Website Purchases" if Pixel not configured)
4. Budget: €120 total, €20/day, 6 days

### Step 2 — Create Ad Sets (2)
**Ad Set A: Andalucía + Murcia**
- Location: Spain → Andalucía, Murcia
- Age: 30-65
- Placements: Facebook feed, Instagram feed, Stories, Reels
- Interests: agricultura, almendro, fruticultura, control de plagas, agricultura sostenible, agricultor, frutales, riego
- Detailed targeting: Expand by location

**Ad Set B: Aragón + Valencia + Castilla-La Mancha**
- Same targeting, different regions

### Step 3 — Create Ads
Upload carousel images (1080×1080 each) + write copy for each card.
For video: Upload MP4 directly, add text overlays in Meta editor or pre-made.
